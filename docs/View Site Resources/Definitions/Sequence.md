# Definition

A sequence is an ordered list of numbers.

## Example

The [[Set|set]] of [[Set|Natural Numbers]] is also a sequence:  $1, 2, 3, 4, 5, 6, ...$

$2, 4, 6, 8, 10, ...$ is the sequence of positive, even [[Set|integers]].
